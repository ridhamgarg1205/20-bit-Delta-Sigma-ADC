# 20-bit-Delta-Sigma-ADC

Designed and implemented a high-resolution Delta-Sigma ($\Delta\Sigma$) ADC targeting future ISRO payload applications, focusing on indigenous VLSI solutions for precise sensing. The project involved developing a Hybrid modulator architecture to optimize jitter robustness and anti-aliasing while achieving 16-19 bits of Effective Number of Bits (ENOB). I engineered a complete digital decimation pipeline, including a 2nd-order CIC filter and a 128-tap FIR filter, and executed the full ASIC design flow from RTL to GDSII using 180 nm technology
